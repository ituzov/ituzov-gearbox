# ituzov-gearbox

Коробка передач для Claude Code: веб-гаджет переключает модель в живой сессии через `tmux send-keys "/model <имя>"`.

## Запуск

- `npm start` (или `node bin/cli.js`). Порт 4321, окно откроется само.
- Claude Code должен работать внутри tmux. Проверка: `tmux list-panes -a -F '#{pane_id} #{pane_current_command} #{pane_title}'`.
- Если tmux-сессии с Claude нет — попроси человека запустить её самому (`tmux`, внутри `claude`). Не запускай его сессию сам.
- Здоровье сервера: `curl http://127.0.0.1:4321/api/panes`.

## Карта проекта

- `bin/cli.js` — точка входа: сервер + открытие окна (Chromium `--app`, отдельный профиль в `~/.gearbox`).
- `src/server.js` — HTTP без зависимостей: `/api/config`, `/api/panes`, `/api/models`, `/api/shift`. Слушает только 127.0.0.1.
- `src/tmux.js` — ЕДИНСТВЕННЫЙ файл с доступом к терминалу: список панелей, автодетект моделей (открыть пикер `/model` → `capture-pane` → Escape), переключение.
- `src/config.js` — дефолтная карта передач + чтение `gearbox.config.json`.
- `public/` — UI: ванильные HTML/CSS/JS, SVG-кулиса, драг с ограничением по прорезям, WebAudio-щелчки, тахометр.

## Правила

- Zero dependencies — не добавляй пакеты без явной просьбы.
- Комментарии в коде и тексты — на русском, живым языком.
- Полная инструкция по установке для агентов — в README, секция «Инструкция для агента».
